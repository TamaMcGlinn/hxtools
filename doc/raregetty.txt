
Syntax: raregetty [-c] [-d defaulthost] [-h host] [-n] tty

  -c    clear screen before displaying "login:"
  -d    specify a default host, which will be used if none is entered
        during login session
  -h    if specified, raregetty limits login to that machine
  -n    do not reopen tty devices (debugging)

A simple login prompt which allows connecting to remote hosts.
